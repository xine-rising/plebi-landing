## Deploy Laravel Backend

```bash
rsync -avz --delete \
  --exclude='.env' \
  --exclude='.git' \
  --exclude='.idea' \
  --exclude='.vscode' \
  --exclude='vendor' \
  --exclude='node_modules' \
  --exclude='storage/app' \
  --exclude='storage/logs' \
  --exclude='database/database.sqlite' \
  --exclude='*.log' \
  --exclude='.phpunit.result.cache' \
  /Users/christinetacha/Desktop/all-hail/ \
  deploy@83.228.215.187:/home/deploy/plebi-backend/
```

Then on the server, in this order:

```bash
cd /home/deploy/plebi-backend

# 1. Install dependencies
composer install --no-dev --optimize-autoloader

# 2. Set up .env FIRST (before any artisan commands)
cp .env.example .env
nano .env  # Edit with production values — see .env checklist below

# 3. Generate app key (only on first deploy)
php artisan key:generate

# 4. Create storage directories
mkdir -p storage/framework/{cache,sessions,views}
mkdir -p storage/app/public
php artisan storage:link

# 5. Set ownership to match PHP-FPM user
# Check which user PHP-FPM runs as:
#   grep -E '^user|^group' /etc/php/*/fpm/pool.d/www.conf
# Then:
sudo chown -R www-data:www-data storage bootstrap/cache

# 6. Run migrations
php artisan migrate --force

# 7. Cache config and routes (AFTER .env is set)
php artisan config:cache
php artisan route:cache
```

## Caddyfile

Before editing, check the PHP-FPM socket path:

```bash
ls /run/php/
# Look for something like php8.5-fpm.sock
# Use the actual filename below
```

Add this block to `/etc/caddy/Caddyfile`:

```caddyfile
plebi.app, www.plebi.app {
    # Laravel API
    handle /api/* {
        root * /home/deploy/plebi-backend/public
        php_fastcgi unix//run/php/php8.5-fpm.sock {
            try_files {path} /index.php?{query}
        }
    }

    # Laravel storage (avatars, etc.)
    handle /storage/* {
        root * /home/deploy/plebi-backend
        file_server
    }

    # Static landing page (catch-all)
    handle {
        root * /home/deploy/plebi-landing/dist
        file_server
        try_files {path} {path}/ /index.html
    }
}
```

**Important:** Validate before reloading. A broken Caddyfile will take down all sites on Juggernaut.

```bash
sudo caddy adapt --config /etc/caddy/Caddyfile
# If no errors:
sudo systemctl reload caddy
```

## Queue Worker (Supervisor)

Create `/etc/supervisor/conf.d/plebi-worker.conf`:

```ini
[program:plebi-worker]
process_name=%(program_name)s_%(process_num)02d
command=php /home/deploy/plebi-backend/artisan queue:work --sleep=3 --tries=3 --max-time=3600
autostart=true
autorestart=true
stopasgroup=true
killasgroup=true
user=deploy
numprocs=1
redirect_stderr=true
stdout_logfile=/home/deploy/plebi-backend/storage/logs/worker.log
stopwaitsecs=3600
```

```bash
sudo supervisorctl reread
sudo supervisorctl update
sudo supervisorctl start plebi-worker:*
```
